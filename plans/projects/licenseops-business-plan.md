# LicenseOps Business Plan (Activation + Entitlement API)

Date: 2026-02-19  
Status: Internal-first, external-later

---

## 1) Why This Project

LicenseOps strengthens all Mintreu products by standardizing:
1. activation
2. entitlement checks
3. revocation
4. domain binding

---

## 2) Product Boundary

Sellable outputs:
1. Hosted LicenseOps API
2. Nuxt admin frontend pack for license operations

Never sell:
1. Core backend source code

---

## 3) Internal Leverage

1. Shopcore and Helpdesk both consume same activation standard.
2. Faster onboarding for new products.
3. Lower support load from license confusion.

---

## 4) External Monetization Path

1. Launch as internal backbone first.
2. Expose as separate API product once stability is proven.
3. Bundle with Shopcore/Helpdesk enterprise tiers.

---

## 5) Priority

1. P0: internal reliability and shared protocol
2. P1: basic external pricing page
3. P2: standalone revenue line
